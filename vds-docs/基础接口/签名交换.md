
# 签名交换

用于获取首个可调用业务接口的签名令牌。

认证与端点统一说明请参考根目录文档：`认证方式与服务器端点`。

## 请求地址

- `POST /api/auth/token`

## 请求体字段

- `appId`：应用 ID（`vap_xxx`），与 `clientId` 二选一
- `clientId`：应用客户端 ID，与 `appId` 二选一
- `clientSecret`：应用密钥

示例：

```json
{
  "appId": "vap_xxxxxxxxxxxxxxxx",
  "clientSecret": "your_app_secret"
}
```

## 响应示例

```json
{
  "accessToken": "eyJhbGciOiJIUzI1NiIs...",
  "apiKey": "eyJhbGciOiJIUzI1NiIs...",
  "tokenType": "Bearer",
  "expiresInSeconds": 3600,
  "appId": "vap_xxxxxxxxxxxxxxxx",
  "grants": ["furtv", "furtv.gathering.timeline"],
  "requestId": "6ff8d966-b3f6-46a6-9fe3-24fd6553ef52"
}
```
