
# 签名请求方式（基础接口）

本页用于索引“基础接口”中的签名相关能力与验证能力。  
默认认证方式与服务器端点统一说明请参考：`认证方式与服务器端点`。

## 接口列表

1. `基础接口/签名交换`  
通过应用凭证换取首个可用签名。

2. `基础接口/签名换新`  
在旧签名进入换新窗口时换取新签名。

3. `基础接口/请求验证接口（HelloWorld）`  
快速验证签名是否可正常调用平台接口。

4. `基础接口/RequestID查询`  
通过 `requestId` 检索对应请求日志结果。

## 推荐阅读顺序

1. 签名交换
2. 请求验证（HelloWorld）
3. 签名换新
4. RequestID 查询
