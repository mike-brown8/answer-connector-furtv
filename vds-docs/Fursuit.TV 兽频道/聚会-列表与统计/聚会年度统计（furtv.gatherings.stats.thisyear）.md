
# 聚会年度统计（furtv.gatherings.stats.thisyear）

获取当前年份聚会总数。

认证与端点统一说明请参考根目录文档：`认证方式与服务器端点`。

## 请求地址

- `GET /api/proxy/furtv/gatherings/stats/this-year`
- 权限节点：`furtv.gatherings.stats.thisyear`

## 成功响应示例

```json
{
  "success": true,
  "total": 127,
  "requestId": "fab0fce5-f4d6-4eb3-8963-6441376728f1"
}
```
